Biểu đồ Bland-Altman¶

BS. Lê Ngọc Khả Nhi¶

Giới thiệu¶

Trong nghin cu y hc, c mt bi ton kh ph bin l kho st v mc tng hp (hay sai bit) gia 2 thit b/cch thc o lng p dng cho cng mt i lng sinh hc. Phng php thng dng l Bland-Altman.

Phng php phn tch ny do 2 nh thng k Bland v Altman xut vo nm 1983, ni dung ca n gm c 3 phn:

Phn gi nh: mc tiu th nht ca quy trnh l kim chng 2 gi nh: 1) Sai bit trung bnh gia 2 kt qu l hng nh trn ton b gii hn ca thang o, v 2) S dao ng (ngu nhin) ca sai bit cng n nh trn ton b thang o.

Phn phn tch nh lng: mc tiu l c tnh 3 tr s: Trung bnh ca sai bit, v 2 gii hn trn/di ca sai bit, c xc nh n gin = mean +/- 1.96 * SD (tng ng 95% mt phn b) vi gi nh sai bit c phn b chun.

Phn ha: Mt biu c dng ln cho php kim tra trc quan 2 gi nh, mc sai bit so vi kt qu ti u (sai bit = 0), v nhn din cc trng hp vt ra khi gii hn trn/di ca sai bit cho php. Biu ny c tn ring l Bland-Altman plot, nhng bn cht ca n n gin l 1 biu tn x c trc honh X biu th trung bnh ca 2 php o (xem nh 1 php c lng thang o chun), v trc tung Y biu th sai bit. Ngi ta thng v thm cc ng thng nh v sai bit = 0, sai bit trung bnh, 2 ngng trn/di ca khong sai bit cho php.

Altman DG, Bland JM (1983). "Measurement in medicine: the analysis of method comparison studies". The Statistician. 32 (3): 307317.

Reinventing the wheel¶

C nhiu bin th ca biu Bland-Altman trong y vn, v trong bi hm nay Nhi cng s to ra mt bin th nh vy, da vo mt s quan im ch quan nh sau:

Thay v biu th trung bnh ca 2 php o, Nhi dng trc honh ch chnh xc 1 php o chun (tham chiu, tiu chun vng).

Nhi s dng 1 biu kt hp (jointplot) gm 3 b phn: 1 biu tn x (c th km mt phn phi 2 chiu - 2D KDE plot) lm trung tm, kt hp vi 2 biu mt phn phi 1 chiu biu th marginal distribution. Cng dng ca nhng KDE plot nhm biu th phn phi thc s ca d liu, vi quan im l i lng cn o v sai bit c th khng theo phn phi Gaussian.

Nhng biu trn c th c t mu theo phn nhm (th d nng ca 1 cn bnh, hay cc phn nhm khc nhau).

Nhi dng trung v v bch phn v th 5+95 thay v trung bnh +/- 1.96 SD, vi cng l do l phn b ca sai bit c th khng chun.

Chng ta cng dng biu ny vi matplotlib v seaborn nh.

Trc ht ta to 1 d liu m phng vi 3 phn nhm A,B,C, 2 kt qu o bi phng php Tiu chun (Ref) v Phng php mc tiu cn kho st (Target), cng khc bit gia chng (Difference)

# Kch thc font ch
plt.rcParams.update({'font.size': 15})

# Thit k li biu  2x2, t l rng 3:1 v cao 1:3 dng GridSpec:

gs = gridspec.GridSpec(2, 2, width_ratios=[3,1], height_ratios=[1,3])
gs.update(hspace=0.09, wspace=0.09)

# To object biu  kch thc 16x10, nn trng

fig = plt.figure(figsize=(12,8))

fig.patch.set_facecolor('white')

# To 2 ph mu lin tc/ri rc cho 3 phn nhm
fill = ['Greens','YlOrRd','Reds'] 
edge = ['#73cc06', '#fcba03', '#fc035a']

# Tnh 3 tr s Median, phn v 5 v 95 ca Difference cho ton qun th
ba_stat = ba_df['Difference'].describe(percentiles = [0.05,0.5,0.95])[4:7].to_list()

# Xc nh gii hn trn v di ca trc honh, trc tung

xmin, xmax = min(ba_df['Ref'])-0.5, max(ba_df['Ref'])+0.5
ymin, ymax = min(ba_df['Difference'])-1, max(ba_df['Difference'])+1

# Thit k h trc cho panel chnh (2D KDE v tn x)
ax = plt.subplot(gs[1,0]) 
ax.set_xlim(xmin, xmax)
ax.set_ylim(ymin, ymax)
ax.grid(True)

ax.set_ylabel('Sai bit 2 php o')
ax.set_xlabel('Thang o tham chiu')

# V KDE plots

for s,fc,ec in zip(ba_df['Class'].unique(), fill, edge):
    x = ba_df[ba_df['Class'] == s]['Ref'].values
    y = ba_df[ba_df['Class'] == s]['Difference'].values
    n = len(x)

    # c tnh kernel density 2D cho mi phn nhm
    
    xx, yy = np.mgrid[xmin:xmax:100j, ymin:ymax:100j]
    positions = np.vstack([xx.ravel(), yy.ravel()])
    values = np.vstack([x, y])
    kernel = st.gaussian_kde(values)
    f = np.reshape(kernel(positions).T, xx.shape)
    
    # V 2D KDE plot (lp nn ca panel chnh)
    
    cfset = ax.contourf(xx, yy, f, levels=5, cmap = fc, alpha = 0.2, extend = 'neither')
    cset = ax.contour(xx, yy, f, colors = ec, levels=5, linewidths=0.5, alpha = 0.5)
    
    # V 2 1D KDE plot cho marginal distribution
    
    axl = plt.subplot(gs[0,0], sharex=ax) # Instantiate right KDE plot area
    axl.get_xaxis().set_visible(False) 
    axl.get_yaxis().set_visible(False)
    axl.spines["right"].set_visible(False)
    axl.spines["left"].set_visible(False)
    axl.spines["top"].set_visible(False)
    axl.spines["bottom"].set_visible(False)
    
    axb = plt.subplot(gs[1,1], sharey=ax) # Instantiate top KDE plot area
    axb.get_xaxis().set_visible(False) # Hide tick marks and spines
    axb.get_yaxis().set_visible(False)
    axb.spines["right"].set_visible(False)
    axb.spines["left"].set_visible(False)
    axb.spines["top"].set_visible(False)
    axb.spines["bottom"].set_visible(False)
    
    # Trc tung
    axb = sns.kdeplot(y, 
                      ax = axb, 
                      color = ec, 
                      shade = True, 
                      alpha = 0.3,
                      vertical = True)
    # trc honh
    axl = sns.kdeplot(x, 
                      ax = axl, 
                      color = ec, 
                      alpha = 0.3,
                      shade = True, 
                      vertical = False)

# V biu  tn x, hin th tr s thng k

for s,ec in zip(ba_df['Class'].unique(), edge):
    x = ba_df[ba_df['Class'] == s]['Ref'].values
    y = ba_df[ba_df['Class'] == s]['Difference'].values
    
    xm = np.median(x)
    ym = np.median(y)
    
    # Tr s cho qun th
    ax.text(xmax + 0.2, ym, f'{ym:.3}', 
            ha='center', 
            va='center',
            color= ec)
    
    axb.hlines(y = ym, xmin = 0, xmax=0.5,
         linestyles = 'dashed',
         alpha = 0.5,
         color = ec)
    
    # Biu  tn x
    sns.scatterplot(x,y,
                ax = ax, 
                alpha = 0.6, 
                color = ec, 
                s = 80,
                label = s)

# Legend panel chnh
ax.legend(loc = 'lower left')

ax.hlines(y = ba_stat + [0],
          xmin = xmin, 
          xmax = xmax,
         linestyles = 'dashed',
         alpha = 0.5)

# Tr s cho phn nhm
for i in ba_stat:
    ax.text(x = xmax - 0.8, 
            y = i,
            s = f'{i:.3}',
            color='k')
